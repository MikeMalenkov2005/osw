# OSW

**OSW** is a simple *Operating System Wrapper* for programming in C.

## C Standard

**OSW** is implemented with **ANSI C** standard with `long long` extension.

## Usage

To include **OSW** in your project add `osw.h` file to your includes and `osw.c`
to your sources. To enable extended *extended functionality* add corresponding
macros to your compiler definitions.

## Extended Functionality

- `OSW_NET` - The networking API extension (stream & datagram sockets).
